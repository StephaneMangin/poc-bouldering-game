# Sprint 8.0 Checklist (5 days)

## Sprint Objective
Build routes R1-R3 with proper gym art direction, wall panels, and color coding.

## Definition of Done
1. R1, R2, R3 playable end-to-end on proper wall panels.
2. Route color coding (green for R1/R2, blue for R3).
3. Hold materials and wall textures (first art pass).
4. Gym environment base (crash pads, basic lighting).
5. Run duration close to route targets.

## Day 1
1. Build wall panel geometry: slab + vertical panels for R1/R2.
2. R1 blockout with hold placement (jugs/open hand only, green color).
3. Start/top hold tag markers.

## Day 2
1. R2 blockout with flag-requiring sections (green color).
2. Build slight overhang panel for R3.
3. R3 blockout with lateral traverse (blue color, crimps + pinches).

## Day 3
1. First art pass: hold materials (colored plastic), wall textures (painted plywood).
2. Crash pad geometry + `PM_CrashPad` material.
3. Basic gym lighting (overhead + wall spots).

## Day 4
1. Tuning pass on routes R1-R3 (duration, rest zones, difficulty).
2. Adjust hold `allowedGripTypes` and `grip_texture` per route.
3. Verify technique requirements match unlock progression.

## Day 5
1. Comparative internal playtest R1-R3.
2. Fix unreadable failure causes.
3. Activate route telemetry.

## Validation Criteria
1. 70% minimum completion on R1 Arcade.
2. Players identify route by color.
3. Green routes feel easier than blue.
4. Gym environment enhances immersion.
