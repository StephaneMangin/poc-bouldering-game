# Sprint 12.0 Checklist (5 days)

## Sprint Objective
Conduct external playtests and apply the most impactful P0/P1 fixes.

## Definition of Done
1. 2 external sessions conducted (novices + challenge players).
2. Summary report with clear prioritization.
3. P0 fixes integrated and verified.
4. Main MVP KPIs close to thresholds.

## Day 1
1. Prepare test build + questionnaire.
2. Verify telemetry (events + logs).
3. Verify all 5 routes completable in all modes.

## Day 2
1. External session A (novices).
2. Collect controls/feedback frictions.
3. Observe physics/audio readability reactions.

## Day 3
1. External session B (challenge players).
2. Collect difficulty/score/technique frictions.
3. Evaluate technique unlock pacing.

## Day 4
1. Sort issues by impact/cost.
2. Implement P0 fixes (controls, physics feel, audio balance).
3. Adjust technique unlock levels if pacing is off.

## Day 5
1. Re-test fixes on both player profiles.
2. Freeze decision for Sprint 13.0.
3. Write playtest report.

## Validation Criteria
1. >= 70% understand the cause of failures.
2. >= 70% want to start another run.
3. No remaining blocking critical bug.
4. Physics model perceived as fair (not arbitrary).
