# Sprint 14.0 Checklist (5 days)

## Sprint Objective
MVP freeze, stability, and demo packaging.

## Definition of Done
1. Stable and presentable MVP build.
2. All critical bugs fixed.
3. Minimum delivery documentation ready.
4. Go/no-go checklist validated.

## Day 1
1. Scope freeze (no new feature).
2. Final triage of remaining bugs.
3. Verify all 5 routes + 3 difficulties + 2 modes stable.

## Day 2
1. Stability and crash fixes.
2. Performance final check (60 FPS target).
3. Memory leak verification on long sessions (>30 min).

## Day 3
1. UX polish: messages, transitions, HUD readability.
2. Audio mix final pass.
3. Visual effects final pass (particles, animations).

## Day 4
1. Release candidate build RC1.
2. Full smoke tests on 5 routes (all modes, all difficulties).
3. Verify save/load persistence (progression, bindings, settings).

## Day 5
1. Final MVP build.
2. Write release notes + next post-MVP priorities.
3. Package demo for distribution.

## Validation Criteria
1. No blocking crash on a 30-minute session.
2. Main MVP KPIs reach the plan's `Go` threshold.
3. Demo showable independently by a new player.
4. All systems (physics, audio, body mechanics, scoring) are cohesive.
