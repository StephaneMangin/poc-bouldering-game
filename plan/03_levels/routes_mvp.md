# MVP Routes

## Objective
Define a set of 5 routes covering learning, technical progression, and final challenge without overloading production.

## MVP Biome
1. Indoor bouldering gym with multiple wall panels (slab, vertical, slight overhang, strong overhang).
2. Hold texture variants (rough, smooth, polished/wet).
3. Light weather/humidity conditions to vary grip.
4. Crash pads below all climbing surfaces.

## Route Color Coding
Following real gym conventions, each route uses a single color for all its holds:
1. `Green`: beginner routes (R1, R2).
2. `Blue`: intermediate routes (R3, R4).
3. `Red`: advanced routes (R5).
4. Non-route holds (features, volumes) remain neutral grey/white.
5. Start holds distinguished by a specific tag marker (tape or tag).
6. Top hold distinguished by a specific tag marker.

## Wall Panel Variety
1. Slab panel (75°-85°): 1 panel, used by R1 and R2.
2. Vertical panel (90°): 2 panels, used by R2 and R3.
3. Slight overhang panel (100°-110°): 1 panel, used by R3 and R4.
4. Strong overhang panel (120°-135°): 1 panel, used by R4 and R5.
5. Roof section (150°+): 1 small section, used by R5 only.

## Route Set
1. `R1 - Vertical Warm-Up` (Easy)
2. `R2 - Short Balance` (Easy)
3. `R3 - Controlled Traverse` (Medium)
4. `R4 - Demanding Compression` (Medium)
5. `R5 - Void Relay` (Hard)

## Details per Route

### R1 - Vertical Warm-Up (Easy)
1. Target duration: 4-6 min.
2. Wall angles: slab + vertical.
3. Focus: learning controls, reading simple holds, basic edging.
4. Grip types: jugs and open hand only.
5. Techniques required: none (basic grab/place).
6. Main risk: wasting endurance unnecessarily.
7. Learning objective: learn to use 1 rest zone.

### R2 - Short Balance (Easy)
1. Target duration: 5-7 min.
2. Wall angles: slab + vertical.
3. Focus: hip placement + footwork + flagging.
4. Grip types: jugs, open hand, 1-2 pinches.
5. Techniques required: flag (L2).
6. Main risk: extended stretched posture.
7. Learning objective: alternate arms to avoid overload.

### R3 - Controlled Traverse (Medium)
1. Target duration: 6-9 min.
2. Wall angles: vertical + slight overhang.
3. Focus: lateral moves and grip management + twist lock.
4. Grip types: jugs, crimps, pinches.
5. Techniques required: twist lock (L3), smearing (L3).
6. Main risk: micro-slips on smooth holds.
7. Learning objective: tactical chalk usage.

### R4 - Demanding Compression (Medium)
1. Target duration: 7-10 min.
2. Wall angles: slight overhang + strong overhang.
3. Focus: technical sequences (heel hook, drop knee).
4. Grip types: crimps, slopers, gastons, underclings.
5. Techniques required: drop knee (L4), heel hook (L5).
6. Main risk: local fatigue in sections without rest.
7. Learning objective: choosing the right moment for a dynamic move.

### R5 - Void Relay (Hard)
1. Target duration: 9-12 min.
2. Wall angles: strong overhang + roof section.
3. Focus: clean execution under pressure, advanced body positioning.
4. Grip types: all types including gastons and underclings.
5. Techniques required: toe hook (L6), knee bar (L7), bicycle (L8).
6. Main risk: low grip + critical fatigue combo.
7. Learning objective: global run optimization.

## Gym Environment Elements
1. Crash pads: continuous soft flooring below all walls.
2. Chalk buckets: placed at the base of each route.
3. Benches: rest seating between walls.
4. Route tags: start/top hold markers per color.
5. Gym lighting: overhead fluorescent + directional spots on featured walls.
6. Background elements: other wall panels visible in distance, gym entrance, water fountain.

## Replayability Variants
1. Weather variant: dry/wet modifies grip loss.
2. Fragile hold variant: positions changed on 2 to 3 segments.
3. Constraint variant: bonus score if focus not used.

## Recommended Unlock Order
1. R1 and R2 available from the start.
2. R3 after 1 completion of R2.
3. R4 after 1 completion of R3.
4. R5 after at least 2 medium completions.

## Route Quality Criteria
1. The route must have at least 2 readable rest zones.
2. Each route must teach one unique main concept.
3. Failure causes must be visually explainable.
4. Median run duration consistent with the route target.
